# jesica12-12.github.io-formulir-pendaftaran-komunitas-teknologi-
Formulir pendaftaran berbasis web sederhana dan responsif. Proyek ini mengimplementasikan HTML5 untuk struktur data dan CSS3 dengan font Poppins untuk styling modern. Formulir ini dirancang untuk mengumpulkan data akun, pribadi (usia, jenis kelamin, minat), dan dokumen portofolio.
